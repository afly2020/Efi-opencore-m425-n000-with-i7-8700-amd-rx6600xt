# Efi-opencore-m425-n000-with-i7-8700-amd-rx6600xt
一、硬件配置：
1、联想台式机M425-N000：主板芯片组为B360m，cpui7-8700，核显UHD630(八代)；原机ps2键盘+USB鼠标
2、原机8G内存，自加8G组成双通道；一块500G容量的ssd装系统；显示器接RX 6600XT DP口
